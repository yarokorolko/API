Projeto API

Bem-vindo ao Projeto API! Este website apresenta 8 APIs diferentes que o Professor pode explorar. Cada API oferece funcionalidades e dados únicos:

OpenWeatherMap API: Fornece dados meteorológicos em tempo real e previsões.

Google Maps API: Oferece serviços de mapas interativos e informações de localização.

Github API: A API do Github permite acessar e interagir com os recursos do Github, como repositórios, usuários, issues e muito mais.

YouTube API: A API do YouTube fornece acesso aos recursos e funcionalidades do YouTube, como vídeos, canais, playlists e comentários.

Summoner API: A API do Summoner (League of Legends) permite acessar informações sobre jogadores, partidas, campeões e estatísticas no jogo League of Legends.

Cats API: A Cats API fornece acesso a imagens e informações relacionadas a gatos.

Rest Countries API: A Rest Countries API oferece informações detalhadas sobre países, incluindo dados demográficos, geográficos, econômicos, culturais e mais.

Random Quotes API: A Random Quotes API fornece citações famosas aleatórias de várias personalidades.
